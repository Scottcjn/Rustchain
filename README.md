## 🏗️ About RustChain

RustChain is an innovative blockchain project that introduces the **Proof-of-Antiquity (PoA)** consensus mechanism. Unlike traditional Proof-of-Work or Proof-of-Stake systems, RustChain rewards users based on the age and historical significance of their CPU hardware, creating a unique ecosystem that preserves computing heritage while securing the blockchain.

### Key Features

- **Proof-of-Antiquity Consensus**: Rewards vintage and historical CPU hardware
- **Sustainable Mining**: Energy-efficient mining that doesn't require modern GPUs
- **Heritage Preservation**: Encourages the preservation of computing history
- **Community-Driven**: Open-source project with active community participation
- **Rust-Based**: Built with Rust for performance and security

## ⚙️ How It Works

The Proof-of-Antiquity consensus mechanism validates blocks based on the age and provenance of CPU hardware:

1. **Hardware Detection**: System analyzes CPU architecture and manufacturing date
2. **Antiquity Scoring**: CPUs receive scores based on age, rarity, and historical significance
3. **Validation**: Higher antiquity scores receive priority in block validation
4. **Rewards**: Miners receive rewards proportional to their hardware's antiquity score

### Supported Architectures

RustChain supports a wide range of vintage CPU architectures including:

- x86 (Intel 8086, 80286, 80386, 80486)
- Motorola 68000 series
- MOS Technology 6502
- Zilog Z80
- And many more historical architectures

## 🚀 Quick Start

### Prerequisites

- Rust 1.70 or higher
- Vintage CPU hardware (optional but recommended for maximum rewards)
- Git

### Installation

```bash
git clone https://github.com/Scottcjn/Rustchain.git
cd Rustchain
cargo build --release
```

### Running the Node

```bash
cargo run --release
```

For detailed installation instructions, see [INSTALL.md](INSTALL.md).
