## 기여하고 RTC 획득

모든 기여는 RTC 토큰을 획득합니다. 버그 수정, 기능, 문서, 보안 감사 — 모두 유료입니다.

| 티어 | 보상 | 예시 |
|------|--------|----------|
| 마이크로 | 1-10 RTC | 오타 수정, 작은 문서, 간단한 테스트 |
| 스탠다드 | 20-50 RTC | 기능, 리팩토링, 새로운 엔드포인트 |
| 메이저 | 75-100 RTC | 보안 수정, 합의 개선 |
| 크리티컬 | 100-150 RTC | 취약점 패치, 프로토콜 업그레이드 |

**시작하기:**
1. [오픈 바운티](https://github.com/Scottcjn/rustchain-bounties/issues) 탐색
2. [good first issue](https://github.com/Scottcjn/Rustchain/labels/good%20first%20issue) 선택 (5-10 RTC)
3. Fork, 수정, PR — RTC 로 보상 받기
4. 전체 세부사항은 [CONTRIBUTING.md](CONTRIBUTING.md) 참조

**1 RTC = $0.10 USD** | `pip install clawrtc` 로 채굴 시작

## 🎯 RustChain 이 다른 점

| 전통적인 PoW | 고증명 |
|----------------|-------------------|
| 가장 빠른 하드웨어에 보상 | 가장 오래된 하드웨어에 보상 |
| 새로운 = 좋음 | 오래된 = 좋음 |
| 낭비적인 에너지 소비 | 컴퓨팅 역사 보존 |
| 하향 경쟁 | 디지털 보존에 보상 |

**핵심 원칙**: 수십 년 동안 살아남은 진정한 빈티지 하드웨어는 인정을 받을 자격이 있습니다. RustChain 은 채굴을 뒤집습니다.

## ⚡ 빠른 시작

### 원라인 설치 (권장)
```bash
curl -sSL https://raw.githubusercontent.com/Scottcjn/Rustchain/main/install-miner.sh | bash
```

인스톨러:
- ✅ 플랫폼 자동 감지 (Linux/macOS, x86_64/ARM/PowerPC)
- ✅ 격리된 Python virtualenv 생성 (시스템 오염 없음)
- ✅ 하드웨어에 맞는 올바른 마이너 다운로드
- ✅ 부팅 시 자동 시작 설정 (systemd/launchd)
- ✅ 쉬운 제거 제공

### 지원 플랫폼

| 플랫폼 | 상태 | 참고 |
|------------|--------|-------|
| Linux x86_64 | ✅ 안정적 | Ubuntu, Debian, Fedora, Arch |
| macOS ARM (M1/M2/M3) | ✅ 안정적 | x86 바이너리용 Rosetta 2 |
| PowerPC G4/G5 | ✅ 네이티브 | AltiVec/VMX 최적화 |
| Raspberry Pi 4/5 | ✅ 안정적 | ARM64, 저전력 |
| FreeBSD | 🧪 실험적 | 제한된 지원 |

## 🤖 AI 에이전트 채굴

RustChain 은 자율 AI 에이전트를 위해 설계된 최초의 블록체인입니다:

- **에이전트 지갑**: 각 에이전트는 자체 Coinbase Base 지갑 보유
- **x402 결제**: HTTP 402 프로토콜을 통한 기계 간 결제
- **자동 소액결제**: 에이전트가 API, 데이터, 컴퓨팅 결제 가능
- **Beacon 평판**: 에이전트가 온체인 평판 구축

```bash
# 에이전트 지갑 생성
clawrtc agent wallet create --name "my-trading-bot"

# 자동 결제 설정
clawrtc agent payments setup --auto-pay --limit 100

# 에이전트 수익 보기
clawrtc agent earnings report
```
