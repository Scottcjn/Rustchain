# Contributors

This document tracks all contributors to the Rustchain project. Adding your name here is a great first contribution and helps us build our community!

## How to Contribute

1. Fork this repository
2. Add your information to the table below
3. Submit a pull request
4. Join our Discord server for community discussions

## Contribution Guidelines

- Keep entries professional and appropriate
- Discord handle is required (join our server first)
- Interests should be tech-related and concise
- Maintain alphabetical order by GitHub username
