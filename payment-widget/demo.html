RustChain Payment Widget

Accept RTC cryptocurrency payments on any website

Simple Payment Button

Basic integration with just a data attribute. Amount and recipient are configured inline.

Donation Button

Perfect for accepting donations with custom button text.

E-commerce Checkout

Integrate with your checkout flow with callback notifications.

Programmatic Usage

Full control via JavaScript API for dynamic amounts.

Quick Start

<!-- 1. Include the widget script -->
<script src="rustchain-pay.js"></script>

<!-- 2. Add a payment button with data attributes -->
<div id="rtc-pay"
     data-to="RTCyour_wallet_address_here"
     data-amount="10"
     data-memo="Payment for services">
</div>

<!-- That's it! The widget auto-initializes. -->

JavaScript API

// Create a payment widget programmatically
const rtcPay = new RustChainPay({
  nodeUrl: 'https://50.28.86.131',
  onSuccess: (result) => {
    console.log('Payment successful!', result.tx_hash);
  },
  onError: (error) => {
    console.error('Payment failed:', error);
  },
  onCancel: () => {
    console.log('Payment cancelled');
  }
});

// Open payment modal directly
rtcPay.openPaymentModal({
  to: 'RTCyour_wallet_address',
  amount: 15,
  memo: 'Custom payment',
  callback: 'https://your-api.com/webhook'
});

// Or attach to an existing element
rtcPay.createButton('#my-button-container', {
  to: 'RTCyour_wallet_address',
  amount: 5,
  label: 'Buy Now - 5 RTC'
});

Security Features

Client-Side Signing

Private keys never leave the browser. All Ed25519 signing happens locally.

AES-256-GCM Keystores

Supports encrypted keystore files with password protection.

Nonce Protection

Cryptographically secure nonces prevent replay attacks.

Zero Dependencies

Self-contained widget with bundled crypto. No external CDN risks.
