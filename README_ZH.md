# RustChain 中文翻译

## 项目概述

RustChain 是一个创新性的区块链项目，专注于通过 Proof-of-Antiquity（古物证明）共识机制来奖励老式硬件设备。

## 核心特性

### 古物证明 (PoA)
- 验证硬件的年代和真实性
- 奖励 vintage 计算设备
- 环保节能的挖矿方式

### RTC 代币
- 项目原生代币
- 用于奖励和交易
- 参与生态系统

## 快速开始

```bash
# 安装
cargo install rustchain

# 运行节点
rustchain-node start

# 创建钱包
rustchain-wallet create
```
