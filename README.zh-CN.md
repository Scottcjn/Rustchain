*你的 PowerPC G4 比现代的 Threadripper 赚得更多。这就是重点。*

## 快速开始

```bash
# 安装
pip install clawrtc

# 创建钱包
clawrtc wallet create

# 开始挖矿
clawrtc mine
```

## 共识机制

RustChain 使用**远古证明 (Proof-of-Antiquity)** 共识：
- 越老的硬件获得越高奖励
- PowerPC G3/G4/G5 有额外乘数
- 现代 x86 CPU 基础奖励较低
