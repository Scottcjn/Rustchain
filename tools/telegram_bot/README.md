# RustChain Telegram Bot

Telegram bot for RustChain community.

## Commands

- `/price` - wRTC price
- `/miners` - Active miners
- `/epoch` - Current epoch
- `/balance <wallet>` - Check balance
- `/health` - Node health

## Setup

```bash
pip install -r requirements.txt
# Edit bot.py and set BOT_TOKEN
python bot.py
```
