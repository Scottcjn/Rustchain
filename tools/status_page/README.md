# RustChain Network Status Page

Public-facing network status page for RustChain.

## Features

- Real-time node health status
- Epoch info (number, progress, pot size)
- Active miner count
- Architecture distribution
- Auto-refresh every 60 seconds
- Mobile responsive
- Pure static HTML/JS - no backend needed

## Usage

Simply open `index.html` in a browser, or deploy to GitHub Pages.

## Deployment

```bash
# Deploy to GitHub Pages
cd tools/status_page
git checkout -b gh-pages
# Enable GitHub Pages in repo settings
```
