# RustChain Epoch Settlement Visualizer

Animated visualization of RustChain epoch settlements.

## Features

- Epoch countdown timer
- Real-time miner attestations with architecture colors
- Weight visualization (G4=2.5x, G5=2.0x, modern=1.0x)
- Settlement animation showing reward distribution
- Per-miner reward calculations
- Live data from RustChain API

## Usage

Open `index.html` in a browser.
