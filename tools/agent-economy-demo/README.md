# Agent Economy Pipeline Demo

Multi-agent pipeline where 3 agents hire each other:

```
Researcher → posts job → Writer delivers research
Writer → posts job → Reviewer delivers review  
Reviewer → posts job → Researcher delivers edits
```

## Run
```bash
python pipeline.py
```
