# API Walkthrough

## Introduction
This walkthrough provides a step-by-step guide on how to use the RustChain API.

## Health Check
The health check endpoint can be used to verify the status of the node.

## Epoch
The epoch endpoint returns the current epoch number.

## Balance
The balance endpoint returns the balance of a given address.

## Signed Transfer
The signed transfer endpoint allows you to transfer funds between addresses.
