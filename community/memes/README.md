# RustChain Retro Computing Meme Gallery

Welcome to the dankest corner of the RustChain ecosystem! This directory hosts our community-driven retro computing meme contest where creativity meets cryptocurrency archaeology.

## Contest Overview

**Active Bounty: 1-3 RTC per meme**
- 1 RTC for each accepted quality meme
- 3 RTC for the top 3 community favorites
- Maximum 5 submissions per contributor

## Submission Process

### 1. Create Your Meme
- **Original content only** - you must create it yourself
- Image format: PNG or JPG, minimum 500px width
- Standard meme templates are encouraged (Drake, distracted boyfriend, expanding brain, etc.)

### 2. Submit via Pull Request
```
community/memes/submissions/[your-username]/
├── meme1.png
├── meme2.jpg
└── submission-info.txt
```

Include a `submission-info.txt` with:
- Brief description of each meme
- Your RTC wallet address for rewards
- Confirmation that content is original

### 3. Community Voting
- Community votes via GitHub reactions on PR
- Voting period: 2 weeks from submission
- Top voted memes featured in gallery

## Content Guidelines

### Must Include At Least One Of:
- Mining cryptocurrency on vintage/legacy hardware
- RustChain / Proof of Antiquity concepts
- PowerPC, G4, G5, POWER8, retro Macs, 486s, classic computers
- The beautiful absurdity of blockchain on ancient machines
- "1 CPU = 1 Vote" democratic mining principles

### Quality Standards
- Clear, readable text
- Professional meme formatting
- Relevant to retro computing + blockchain themes
- Family-friendly content
- Respectful humor (no offensive content)

## Reward Distribution

### Automatic Rewards (1 RTC each)
- Memes meeting quality standards
- Original content verification passed
- Follows community guidelines
- Minimum 3 positive community reactions

### Top 3 Bonus (3 RTC each)
- Highest community vote count
- Exceptional creativity or humor
- Strong community engagement
- Represents RustChain values
