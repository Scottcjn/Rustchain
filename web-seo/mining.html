RustChain Mining Guide

Start mining RustChain (RTC) with vintage hardware in just a few minutes. The clawrtc mining client automatically detects your hardware, validates authenticity, and connects you to the RustChain network.

Requirements

Operating System

  • Linux (Ubuntu, Debian, Fedora, Arch, etc.)
  • macOS 10.13+ (for Apple Silicon or Intel Macs)
  • Windows 10/11 (with WSL2 recommended)

Software

  • Python 3.11 or higher
  • pip (Python package manager)
  • Internet connection

Hardware

  • Recommended: Vintage hardware (PowerPC, 68K, SPARC, retro x86)
  • Supported: Any authentic computing hardware
  • Note: Virtual machines and emulators are not allowed

Quick Start

1

Install clawrtc

pip install clawrtc

This installs the official RustChain mining client.

2

Run the Miner

clawrtc

The client will automatically detect and configure your hardware.

3

Start Mining

Your hardware will begin attesting and earning rewards.

First epoch payout after ~24 hours.

Advanced Configuration

For more control over mining behavior, clawrtc supports several options:

Option Description Example
--node-url Custom RustChain node URL clawrtc --node-url https://node.rustchain.org:8099
--wallet Custom wallet address clawrtc --wallet 0x1234...
--pool Mine via mining pool clawrtc --pool http://pool.rustchain.org:8080
--verbose Detailed logging output clawrtc --verbose

Mining Pools

If you have weaker vintage hardware or want more consistent rewards, you can join a RustChain mining pool:

  • Combine Power: Multiple miners share computing power
  • Consistent Rewards: More predictable payout schedule
  • Lower Barrier: Even small miners can earn regularly

To mine via pool: clawrtc --pool http://pool.rustchain.org:8080

Troubleshooting

Hardware Not Detected

Run with --verbose to see detailed hardware detection info.

Connection Refused

Check your internet connection and verify the node URL is accessible.

VM Detected Error

Virtual machines are not allowed. Use authentic vintage hardware.

Low Rewards

Modern hardware earns lower multipliers. Use vintage hardware for better rewards.

Withdraw Your Earnings

Once you've accumulated enough RTC, you can withdraw to an external wallet:

  1. Register your withdrawal address
  2. Request withdrawal (minimum: 0.1 RTC)
  3. Wait for processing (usually within one epoch)
  4. Receive RTC in your external wallet
