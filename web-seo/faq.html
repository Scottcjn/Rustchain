What is RustChain?

RustChain is a blockchain that rewards vintage hardware mining through a unique Proof of Antiquity consensus mechanism. Unlike traditional cryptocurrencies that favor modern hardware, RustChain celebrates computing history by rewarding older and rarer hardware with higher multipliers.

The network aims to reduce e-waste by giving vintage computers a purpose: mining cryptocurrency while preserving computing heritage.

How do I start mining?

  1. Install Python 3.11+ if you don't have it
  2. Install clawrtc: pip install clawrtc
  3. Run the miner: clawrtc
  4. Wait for rewards: Payouts occur every epoch (24 hours)

üí° Tip: Run clawrtc --verbose to see detailed hardware detection information.

What are the system requirements?

Operating System: Linux, macOS 10.13+, or Windows 10/11 (WSL2 recommended)
Software: Python 3.11+ and pip
Hardware: Any authentic computing hardware (vintage recommended)
Network: Stable internet connection

What hardware earns the most?

Vintage hardware earns significantly more due to higher multipliers:

  • 68K Macintosh: 3.0x (highest)
  • PowerPC G4: 2.5x
  • PowerPC G5: 2.0x
  • SPARC: 2.0x
  • PowerPC G3: 1.8x
  • 486 / Pentium: 1.8x - 2.5x
  • Modern Hardware: 1.0x (base)

Can I mine with a virtual machine (VM)?

No. RustChain uses hardware fingerprinting and anti-emulation checks to detect virtual machines. VMs, emulators, and cloud instances will be detected and rejected.

How do mining pools work?

Mining pools allow multiple miners to combine their computing power and receive more consistent rewards:

  • Miners connect to the pool instead of mining directly
  • Pool submits combined attestations to the network
  • Rewards are distributed proportionally based on contribution
  • Pool charges a small fee (usually 1%) for operating costs

clawrtc --pool http://pool.rustchain.org:8080

How long until I get my first payout?

Payouts occur every epoch, which is 24 hours. Your first payout will be distributed approximately 24 hours after your first successful attestation.

üí∞ RTC Tokens

What is the RTC token?

RTC is the native cryptocurrency of RustChain. It's earned through mining proof of antiquity and can be:

  • Withdrawn to external wallets
  • Traded on decentralized exchanges (DEX)
  • Bridged to other blockchains (Ergo, Solana)
  • Used for protocol governance (future)

How do I withdraw my RTC?

  1. Register your withdrawal address on the node
  2. Request withdrawal (minimum: 0.1 RTC)
  3. Pay fee (0.01 RTC)
  4. Wait for processing (usually within one epoch)

üìä Daily Limit: 1,000 RTC maximum withdrawal per day

What are the tokenomics?

Total Supply: 21,000,000 RTC (fixed)
Daily Emission: 1.5 RTC per epoch
Halving: Rewards halve approximately every 4 years
Distribution: 70% to miners, 30% to development, marketing, and reserve

What is Proof of Antiquity?

Proof of Antiquity is a consensus mechanism that rewards hardware based on age and rarity. It uses:

  • Hardware Fingerprinting: CPU architecture, ROM verification, thermal profiling
  • Entropy Scoring: Timing characteristics unique to hardware
  • Anti-Emulation: Detection of VMs and emulators

This ensures only authentic vintage hardware can mine, creating a fair distribution that favors preservation over raw computing power.

How is RustChain different from Bitcoin?

Feature Bitcoin RustChain
Consensus Proof of Work Proof of Antiquity
Mining ASICs dominate Vintage hardware favored

Is RustChain secure?

Yes. RustChain includes multiple security measures:

  • Hardware Authentication: Fingerprinting prevents spoofing
  • Anti-Cheating: Timing analysis detects simulation
  • No VM Mining: Emulators are detected and rejected
  • Transparent: All data is publicly verifiable on-chain
  • Replay Protection: Prevents double-spending attacks

üîß Troubleshooting

Hardware not detected correctly?

Try running with verbose mode to see detailed detection:

clawrtc --verbose

This will show CPU architecture, model, and fingerprint detection details. If detection fails, check that you're using authentic hardware (not a VM).

Connection refused to node?

  1. Check your internet connection
  2. Verify the node URL is correct
  3. Check if a firewall is blocking the connection
  4. Try a different node URL

clawrtc --node-url https://node.rustchain.org:8099

Low rewards received?

Check the following:

  • Verify your hardware multiplier (modern hardware earns 1.0x)
  • Ensure you're mining consistently (uptime bonus)
  • Check attestation frequency (regular attestations earn more)
  • Review epoch statistics on the network
