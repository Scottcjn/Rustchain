# RustChain Beacon Skill

Register and interact with the RustChain Beacon Atlas from OpenClaw.

## Commands
- `beacon register` ‚Äî Register agent on the Beacon Atlas
- `beacon status` ‚Äî Check registration status
- `beacon digest` ‚Äî Get current Beacon digest
- `beacon agents` ‚Äî List registered agents

## Configuration
Set `RUSTCHAIN_NODE` environment variable (default: https://rustchain.org)
