# Документация RustChain

> **RustChain** — это блокчейн Proof-of-Antiquity, который вознаграждает старое оборудование более высокими множителями майнинга. Сеть использует 6 проверок аппаратного отпечатка для предотвращения получения вознаграждений виртуальными машинами и эмуляторами.

## Основная сеть

- **Основной узел**: `https://rustchain.org`
- **Исследователь**: `https://rustchain.org/explorer/`
- **Проверка работоспособности**: `curl -fsS https://rustchain.org/health`
- **Страница статуса сети**: `docs/network-status.html` (панель статуса, размещаемая на GitHub Pages)

## Текущие статистические данные

```bash
# Проверить работоспособность узла
curl -fsS https://rustchain.org/health | jq .

# Список активных майнеров
curl -fsS https://rustchain.org/api/miners | jq .

# Информация о текущей эпохе
curl -fsS https://rustchain.org/epoch | jq .
```

## Обзор архитектуры

```
┌─────────────────┐     ┌──────────────────┐     ┌─────────────────┐
│  Vintage Miner  │────▶│ Attestation Node │────▶│  Ergo Anchor    │
│  (G4/G5/SPARC)  │     │  (rustchain.org)  │     │ (Immutability)  │
└─────────────────┘     └──────────────────┘     └─────────────────┘
        │                        │
        │ Hardware Fingerprint   │ Epoch Settlement
        │ (6 checks)             │ Hash
        ▼                        ▼
   ┌─────────┐              ┌─────────┐
   │ RTC     │              │ Ergo    │
   │ Rewards │              │ Chain   │
   └─────────┘              └─────────┘
```
