# Краткое руководство по RustChain

Пошаговое руководство для новых пользователей. Каждая команда готова к копированию и вставке.

---

## Что такое RustChain?

RustChain — это блокчейн, который вознаграждает вас за поддержание старых компьютеров в рабочем состоянии. Вместо того чтобы награждать самую быструю машину (как Bitcoin), RustChain награждает *самую старую* машину. Токен называется **RTC** (RustChain Token), и он имеет реальную ценность — 1 RTC стоит примерно $0.10 USD. Более 260 контрибьюторов заработали 25 000+ RTC через майнинг и баунти за код.

---

## Предварительные требования

Вам понадобятся две вещи:

- **Компьютер** — буквально любой компьютер. Linux, macOS, Windows, Raspberry Pi, PowerPC Mac, даже SPARC-сервер. Если он запускает Python, он может майнить.
- **Только macOS:** Python 3 через Homebrew — перед установщиком выполните `brew install python3`.
- **Интернет-соединение** — ваш майнер общается с сетью RustChain, чтобы доказать подлинность вашего оборудования.

Это всё. GPU не требуется. Регистрация аккаунта не нужна.

## Шаг 1: Установка майнера

Откройте терминал в Linux или macOS и выполните:

```bash
curl -sSL https://raw.githubusercontent.com/Scottcjn/Rustchain/main/install-miner.sh | bash
```

### Предварительное условие Homebrew для macOS

Установщик использует системный `python3`. На свежей установке macOS сначала установите Python через Homebrew:

```bash
brew update
brew install python3
python3 --version
```

Затем выполните команду установщика RustChain выше.

В Windows вместо команды Bash используйте установщик майнера для Windows. См. `miners/windows/installer/README.md` и запустите `miners/windows/rustchain_miner_setup.bat` из пакета майнера Windows.

**Что это делает:**

1. Определяет вашу операционную систему и архитектуру процессора.
2. Устанавливает Python 3, если у вас его нет (только Linux — пользователи macOS должны сначала выполнить `brew install python3`; Windows — предустановлен).
3. Загружает скрипт майнера в `~/.rustchain/`.
4. Создает виртуальное окружение Python с зависимостями.
5. Запрашивает у вас имя кошелька.
6. Настраивает майнер для автоматического запуска при загрузке.
7. Тестирует соединение с сетью RustChain.

**Хотите предварительно просмотреть без установки?** Добавьте `--dry-run`:

```bash
curl -sSL https://raw.githubusercontent.com/Scottcjn/Rustchain/main/install-miner.sh | bash -s -- --dry-run
```

### Выбор имени кошелька

Во время установки вы увидите:

```
[?] Введите имя кошелька (или нажмите Enter для автогенерации):
```

Введите имя, которое вы запомните, например `scott-laptop` или `my-g4-mac`. Это ваш адрес кошелька — так вы будете получать RTC. Если вы нажмете Enter, не вводя ничего, установщик сгенерирует его автоматически (например, `miner-myhost-4821`).

**Запишите имя вашего кошелька.** Оно вам понадобится для проверки баланса позже.

### Установка с указанием имени кошелька (пропуск запроса)

```bash
curl -sSL https://raw.githubusercontent.com/Scottcjn/Rustchain/main/install-miner.sh | bash -s -- --wallet my-cool-wallet
```

## Шаг 2: Проверка установки

После завершения установки убедитесь, что всё на месте:

```bash
ls ~/.rustchain/
```

Вы должны увидеть:

```
rustchain_miner.py      # Скрипт майнера
fingerprint_checks.py   # Модуль проверки оборудования
start.sh                # Скрипт быстрого запускаenv/                   # Виртуальное окружение Python
```

Проверьте доступность сети:

```bash
curl -sk https://rustchain.org/health
```

Вы должны увидеть что-то вроде:

```json
{
  "ok": true,
  "version": "2.2.1-rip200",
  "uptime_s": 3966,
  "db_rw": true
}
```

Если появляется `"ok": true`, сеть онлайн, и ваша машина может с ней связаться.

## Шаг 3: Запуск майнинга

Если установщик настроил автозапуск (по умолчанию он это делает), ваш майнер уже работает. Проверьте его статус:

**Linux:**

```bash
systemctl --user status rustchain-miner
```

**macOS:**

```bash
launchctl list | grep rustchain
```

### Ручной запуск (при необходимости)

```bash
~/.rustchain/start.sh
```

Или запустите майнер напрямую:

```bash
~/.rustchain/venv/bin/python ~/.rustchain/rustchain_miner.py --wallet ИМЯ_ВАШЕГО_КОШЕЛЬКА
```

### Что вы увидите

Когда майнер запустится, он выполнит 6 проверок аппаратного отпечатка, чтобы доказать, что ваша машина реальна (а не виртуальная машина):

```
[1/6] Clock-Skew & Oscillator Drift... PASS
[2/6] Cache Timing Fingerprint... PASS
[3/6] SIMD Unit Identity... PASS
[4/6] Thermal Drift Entropy... PASS
[5/6] Instruction Path Jitter... PASS
[6/6] Anti-Emulation Checks... PASS

OVERALL RESULT: ALL CHECKS PASSED
```

Затем он начнет каждые несколько минут отправлять аттестации (подтверждая ваше оборудование) в сеть. Вы увидите строки логов вида:

```
[+] Attestation accepted. Next attestation in 300s.
```

Это означает, что ваш майнер работает. Оставьте его включенным.

## Шаг 4: Проверка баланса

Вознаграждения распределяются каждые **10 минут** (одна «эпоха»). После завершения вашей первой эпохи проверьте баланс:

```bash
curl -sk "https://rustchain.org/wallet/balance?miner_id=ИМЯ_ВАШЕГО_КОШЕЛЬКА"
```

Замените `ИМЯ_ВАШЕГО_КОШЕЛЬКА` на имя кошелька, которое вы выбрали при установке. Пример:

```bash
curl -sk "https://rustchain.org/wallet/balance?miner_id=scott-laptop"
```

Ответ:

```json
{
  "miner_id": "scott-laptop",
  "balance_rtc": 0.119051
}
```

Эти `0.119` RTC — ваше первое вознаграждение за майнинг. Оно будет продолжать расти, пока майнер работает.

### Проверка в обозревателе блоков

Вы также можете увидеть всю сеть, всех майнеров и ваши вознаграждения здесь:

[https://rustchain.org/explorer/](https://rustchain.org/explorer/)

## Шаг 5: Понимание вашего заработка

Каждые 10 минут 1.5 RTC распределяются между всеми активными майнерами. Ваша доля зависит от **множителя древности** вашего оборудования — старое оборудование получает больший кусок.

### Таблица множителей оборудования

| Оборудование | Множитель | Пример |
|---|---|---|
| DEC VAX, Inmos Transputer | 3.5x | Музейное железо |
| Motorola 68000 | 3.0x | Amiga, классический Mac |
| Sun SPARC | 2.9x | Рабочая станция класса люкс |
| PowerPC G4 | **2.5x** | Эпоха PowerBook, iBook, Power Mac |
| PowerPC G5 | **2.0x** | Башни Power Mac G5 |
| PowerPC G3 | 1.8x | Эпоха Bondi Blue iMac |
| IBM POWER8 | 1.5x | Корпоративные серверы |
| Pentium 4 | 1.5x | Начало 2000-х |
| RISC-V | 1.4x | Открытое оборудование, будущее |
| Apple Silicon (M1-M4) | 1.2x | Современные, но приветствуются |
| Современный x86 (AMD/Intel) | 0.8x | Базовый уровень |
| ARM NAS/SBC | 0.0005x | Слишком дешевые, слишком уязвимы для ферм |

**Есть PowerBook G4, пылящийся в шкафу?** Подключите его. Он зарабатывает в 2.5 раза больше, чем ваш игровой ПК.

### Примерный заработок (8 майнеров онлайн)

```
PowerPC G4 (2.5x):       0.30 RTC/epoch
PowerPC G5 (2.0x):       0.24 RTC/epoch
Современный x86 ПК (0.8x):    0.12 RTC/epoch
```

За 24 часа (144 эпохи) Mac G4 заработает примерно **43 RTC** ($4.30), в то время как современный
ПК заработает примерно **17 RTC** ($1.70). Больше майнеров в сети — меньше индивидуальные доли, но и более здоровая сеть.

## Шаг 6: Зарабатывайте больше с баунти

Майнинг — это пассивный доход. Для более крупных выплат вносите вклад кодом.

### Просмотр открытых баунти

[https://github.com/Scottcjn/rustchain-bounties/issues](https://github.com/Scottcjn/rustchain-bounties/issues)

Каждый Issue с меткой bounty имеет указанное вознаграждение в RTC. Вознаграждения варьируются от 1 RTC (исправление опечатки) до 200 RTC (уязвимость безопасности).

| Уровень | Вознаграждение | Примеры |
|---|---|---|
| Микро | 1-10 RTC | Исправить опечатку, улучшить документацию, добавить тест |
| Стандартный | 20-50 RTC | Новая функция, рефакторинг, интеграция |
| Основной | 75-100 RTC | Исправление безопасности, улучшение протокола |
| Критический | 100-200 RTC | Обнаружение уязвимости, работа над консенсусом |

### Как получить баунти

1. Найдите Issue с баунти, над которым хотите поработать.
2. Оставьте комментарий под Issue с именем вашего кошелька (чтобы мы знали, куда вам платить).
3. Сделайте форк репозитория и отправьте Pull Request.
4. Как только ваш PR будет рассмотрен и принят, RTC будет отправлен на ваш кошелек.

### Самый простой первый вклад

Ищите Issue с меткой `good first issue` или предложите улучшение документации. Даже исправление одной опечатки в README принесет вам RTC.

## Шаг 7: Просмотр сети

### Живой обозреватель

Посмотрите всех майнеров, блоки и балансы здесь:

[https://rustchain.org/explorer/](https://rustchain.org/explorer/)

### Конечные точки API (для любопытных)

Все они работают из вашего терминала:

```bash
# Сеть жива?
curl -sk https://rustchain.org/health

# Кто сейчас майнит?
curl -sk https://rustchain.org/api/miners

# В какой мы эпохе?
curl -sk https://rustchain.org/epoch

# Какой у меня баланс?
curl -sk "https://rustchain.org/wallet/balance?miner_id=ИМЯ_ВАШЕГО_КОШЕЛЬКА"
```

Флаг `-sk` говорит curl принять самоподписанный TLS-сертификат. Это нормально — узел использует самоподписанный сертификат, а не коммерческий.

## Устранение неполадок

### `ConnectionRefused` или «Не удается подключиться к узлу bootstrap»

Обычно это означает, что ваша машина не может достичь узла RustChain.

1. Проверьте, отвечает ли основной узел:

```bash
curl -sk https://rustchain.org/health
```

2. Если это не удается, подождите 30-60 секунд и повторите попытку. Узел может перезапускаться.
3. Убедитесь, что ваш интернет-соединение, брандмауэр, VPN или прокси не блокируют исходящие HTTPS-соединения.
4. Если вы установили пользовательский URL узла, проверьте имя хоста, порт и схему.

### `InsufficientBalance`

Майнинг-вознаграждения не требуют платного аккаунта, но некоторые действия кошелька или моста могут потребовать существующего баланса RTC для оплаты комиссий.

1. Убедитесь, что вы используете точное имя кошелька из установки:

```bash
curl -sk "https://rustchain.org/wallet/balance?miner_id=ВАШЕ_ТОЧНОЕ_ИМЯ_КОШЕЛЬКА"
```

2. Подождите как минимум одну полную эпоху после первого запуска майнера. Вознаграждения начисляются примерно каждые 10 минут.
3. Если вы тестируете действие кошелька до получения вознаграждений, обратитесь за помощью к сообществу или используйте фaусет/тестнет, когда они будут доступны.

### `HardwareFingerprintMismatch`

Это может произойти после обновлений BIOS, прошивки, изменений в VM/контейнерах или перемещения майнера между разным оборудованием.

1. Запустите майнер на голой машине, а не внутри VM или контейнера.
2. Перезапустите майнер, чтобы он выполнил новую аттестацию.
3. Если вы недавно обновляли BIOS или прошивку, считайте машину измененным профилем оборудования и повторите процесс установки/аттестации с тем же именем кошелька.

### Контрольный список конфигурации майнера

- Имя кошелька в вашей команде соответствует кошельку, на который вы хотите получить оплату.
- `curl -sk https://rustchain.org/health` возвращает `"ok": true`.
- Ваше системное время установлено правильно; окна TLS и аттестации могут завершиться неудачно, когда время сильно отстает.
- Вы работаете на реальном оборудовании, если ожидаете нормальных наград.
- Вы подождали как минимум 2-3 эпохи, прежде чем решили, что награды отсутствуют.

### «Python 3 не найден»

Установщик пытается установить Python автоматически в Linux. В macOS или Windows вам нужно сначала установить его самостоятельно:

- **macOS:** `brew install python3` (или загрузите с https://python.org)
- **Windows:** Загрузите с https://python.org/downloads и отметьте «Add to PATH»

### «curl: command not found»

- **Linux:** `sudo apt install curl` (Debian/Ubuntu) или `sudo dnf install curl` (Fedora)
- **macOS:** curl предустановлен на всех Mac.

### Ошибки SSL-сертификата

Если вы видите ошибки, связанные с сертификатами, при выполнении команд `curl`, добавьте `-k`:

```bash
curl -sk https://rustchain.org/health
```

Скрипт майнера обрабатывает это автоматически.

### Майнер запускается, но наград нет через 30 минут

1. Убедитесь, что ваш майнер отображается в списке активных майнеров:

```bash
curl -sk https://rustchain.org/api/miners
```

Ищите имя вашего кошелька в выводе.

2. Убедитесь, что вы запрашиваете правильное имя кошелька:

```bash
curl -sk "https://rustchain.org/wallet/balance?miner_id=ВАШЕ_ТОЧНОЕ_ИМЯ_КОШЕЛЬКА"
```

3. Вознаграждения начисляются каждые 10 минут. Подождите как минимум 2-3 эпохи (20-30 минут).

### Виртуальные машины получают почти нулевое вознаграждение

Это предусмотрено дизайном. VM (VMware, VirtualBox, QEMU, WSL) обнаруживаются с помощью проверки отпечатка против эмуляции и получают примерно 1/миллиардную часть от нормальных вознаграждений. RustChain вознаграждает только реальное оборудование. Запускайте майнер на голой машине, а не внутри VM.

### Удаление

Чтобы полностью удалить майнер:

```bash
curl -sSL https://raw.githubusercontent.com/Scottcjn/Rustchain/main/install-miner.sh | bash -s -- --uninstall
```

## Глоссарий

| Термин | Значение |
|---|---|
| **RTC** | RustChain Token — криптовалюта, которую вы зарабатываете майнингом. 1 RTC стоит примерно $0.10 USD. |
| **Эпоха** | 10-минутный интервал. В конце каждой эпохи 1.5 RTC распределяется между всеми активными майнерами. |
| **Аттестация** | Процесс, в ходе которого ваш майнер доказывает подлинность своего оборудования, выполняя 6 проверок отпечатка. |
| **Множитель древности** | Бонус, основанный на возрасте вашего оборудования. Старые процессоры получают более высокие множители. |
| **Кошелек** | Имя/адрес вашего майнера. Сюда отправляются ваши RTC. Вы выбрали его при установке. |
| **Майнер** | Программное обеспечение, работающее на вашей машине, которое подтверждает данные сети и зарабатывает RTC. |
| **Отпечаток** | 6 измерений аппаратного обеспечения (дрейф тактовой частоты, время отклика кэша, идентификатор SIMD, тепловой дрейф, дрожание инструкций, проверки против эмуляции), которые доказывают реальность вашей машины. |
| **wRTC** | Обернутый RTC на Solana. Вы можете обменять RTC на wRTC, используя мост на bottube.ai/bridge. |
| **Обозреватель блоков** | Веб-страница, отображающая всю сетевую активность: майнеров, балансы, эпохи. Посетите rustchain.org/explorer. |
