# RustChain: The Blockchain That Rewards Old Computers

## What if your grandma's old PC was worth more than your gaming rig?

That's the premise behind **RustChain** — a blockchain where a 1999 Pentium III earns more cryptocurrency than a 2024 RTX 4090. It sounds backwards, but it's the future of sustainable mining.

---

## What is RustChain?

RustChain is an independent blockchain that runs a consensus mechanism called **Proof of Antiquity**. Unlike Bitcoin's Proof of Work (which rewards raw computing power) or Ethereum's Proof of Stake (which rewards capital), RustChain rewards **hardware age and authenticity**.

The token is called **RTC** (RustChain Token). It's earned by running a lightweight miner on real hardware — the older and rarer your CPU, the more you earn per epoch.

**Key stats:**
- 5 attestation nodes across 3 continents
- 24+ active miners on real vintage hardware
- CPU-only mining (zero GPU impact, <0.1% overhead)
- Free to start — just install Python and run the miner

---

## Proof of Antiquity: How It Works

Traditional blockchains are a race to waste electricity. RustChain flips that model:

1. **Hardware Fingerprinting** — Your miner generates a unique fingerprint based on CPU timing characteristics, architecture detection, and device entropy
2. **Attestation** — The fingerprint is submitted to a RustChain node, which verifies the hardware is real vintage silicon
3. **Multiplier Calculation** — Your CPU's age determines your earning multiplier:
   - Intel 386 (1985): **3.0x**
   - Pentium III (1999): **2.0x**
   - PowerPC G4 (2003): **2.5x**
   - Modern Core i9 (2024): **0.8x**

The network literally pays you more for preserving computing history.

### Why This Matters

- **E-waste reduction** — Old computers become revenue-generating assets instead of landfill
- **Historical preservation** — Vintage hardware gets a second life
- **Energy efficiency** — Mining uses CPU cycles, not GPU farms
- **Accessibility** — Anyone with an old computer can participate

## Getting Started in 5 Minutes

### Option 1: One-Line Install

```bash
curl -fsSL https://rustchain.org/install.sh | bash -s -- --wallet YOUR_WALLET
```

### Option 2: Python Install

```bash
pip install clawrtc
clawrtc mine --wallet YOUR_WALLET
```

### Option 3: Vintage Hardware (The Fun Part)

Got an old computer? Install Linux on it and mine:

```bash
# On your old Pentium 4 or PowerPC Mac
sudo apt-get install python3 python3-pip
pip3 install clawrtc
clawrtc mine --wallet grandmas-pentium-iii
```

Your hardware will be fingerprinted and you'll start earning RTC with your antiquity multiplier.

### Check Your Earnings

```bash
clawrtc balance --wallet YOUR_WALLET
# Or visit: https://rustchain.org/explorer/
```
