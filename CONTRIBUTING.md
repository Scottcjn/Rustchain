# Contributing to RustChain

Thank you for your interest in contributing to RustChain - the Proof-of-Antiquity blockchain!

## Quick Start

1. **Fork the repository**
2. **Clone your fork:**
   ```bash
   git clone https://github.com/YOUR_USERNAME/Rustchain.git
   cd Rustchain
   ```
3. **Create a branch:**
   ```bash
   git checkout -b feature/your-feature
   ```
4. **Make your changes**
5. **Test thoroughly**
6. **Commit:**
   ```bash
   git commit -m "feat: your change"
   ```
7. **Push:**
   ```bash
   git push origin feature/your-feature
   ```
8. **Open a Pull Request**

## Code Style

- Follow Rust idioms and best practices
- Write clear commit messages
- Add tests for new features
- Update documentation as needed

## PR Guidelines

- One feature/fix per PR
- Link related issues (e.g., "Closes #123")
- Describe your changes clearly
- Include testing steps

## Bounty Tasks

If you're working on a bounty task:
1. Comment on the issue to claim it
2. Include your wallet address in the PR description
3. RTC will be transferred after merge

## Questions?

Open an issue for discussion before starting major changes.
