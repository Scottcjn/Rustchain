# Contributing to Rustchain

We welcome contributions from everyone!

## Setup

```bash
git clone https://github.com/Scottcjn/Rustchain.git
cd Rustchain
cargo build
```

## Testing Your Changes

Run the full test suite:

```bash
cargo test --all
```

Or run specific test modules:

```bash
cargo test -p rustchain-core
cargo test -p rustchain-consensus
cargo test -p rustchain-network
```

## Code Style

- Follow Rust standard formatting (`cargo fmt`)
- Run clippy before submitting (`cargo clippy`)
- Keep commits organized and descriptive

## Submitting a Pull Request

1. Fork the repo
2. Create a feature branch
3. Make your changes
4. Run the tests
5. Submit a PR with description of changes

## Need Help?

Create a discussion or reach out via issues.
