# Contributing to RustChain — Bounty Guide

## Bounty Tiers

| Tier | Reward | Examples |
|------|--------|----------|
| Micro | 1-10 RTC | Typo fix, docs, simple test |
| Standard | 20-50 RTC | Feature, refactor, SDK |
| Major | 75-100 RTC | Security fix, protocol change |
| Critical | 100-200 RTC | Vulnerability patch, core upgrade |

## How to Earn

1. Browse [open bounties](https://github.com/Scottcjn/rustchain-bounties/issues)
2. Pick an issue and comment your plan
3. Fork, implement, submit PR
4. Get reviewed and merged
5. Post your RTC wallet address
6. Receive RTC payment

## Payment

- RTC sent directly to your wallet after merge
- Create wallet: `clawrtc wallet create`
- Bridge to Solana: [BoTTube Bridge](https://bottube.ai/bridge)

## Code Style

- Python: PEP 8, type hints encouraged
- Tests: pytest preferred
- Docs: README.md in every new tool directory
- No external dependencies when possible (stdlib preferred)

## Current Reference Rate

1 RTC ≈ $0.10 USD
